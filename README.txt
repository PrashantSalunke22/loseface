  ========================================
  INTRODUCTION
  ========================================

  This is a face recognition project aimed to build a complete
  functional system to detect subjects.


  ========================================
  STATUS
  ========================================

  This project is under development. Do not expect to have an
  user-friendly interface in some months.


  ========================================
  GETTING SOURCE CODE
  ========================================

  You can get the last copy of source code from SVN

    svn co https://loseface.svn.sourceforge.net/svnroot/loseface/trunk loseface

  Code in SVN is under development, so there are big chances
  to be unstable.
